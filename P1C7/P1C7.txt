=Part 1 Chapter 7: EXPRESSIONS=

-Operators-
Operators do work on arguments called operands.

-Logical Operators-
Regular operators.
Bitwise logical operators. (Perform a boolean operation at the bit level and return an integral type matching the operands).

-Arithmetic Operators-
Unary and binary arithmetic operators work with integral and floating-point types. Used for mathetmatical computations.

-Binary Arithmetic Operators-
Five: Addition, subtraction, multiplication, division, and modulo. Promotion on the operands. Double + long double = long double, etc.

If one argument is signed, both operands become signed. Size of the largest operand is used to promote the other operand.

Promotion is the set of rules for interpreting literals.

-Assignment Operators-
Performs a given operation and assigns the result to the first operand.

```
x = y;
```

Promotion rules don't apply.

-Increment and Decrement Operators-
```
++x;
```

-Comparison Operators-
Evaluate to bool.

```
x==y;
```

-Member Access Operators-
Interact with pointers, arrays, and many classes.

```
*x; // pointer-to-member-of-object
```

Provides access to the element pointed to by x.

-Ternary Conditional Operator-
```
x ? y : z;
```
x is evaluated as a boolean expression and returns the second operand or the third.

-The Comma Operator-
Allows several expressions separated by commas to be evaluated withing a larger expression, evaluating from left to right. The rightmost is the return value.

```
return x = 9, x++, x / 2;
```

-Operator Overloading-
```
FunctionName operator+(unsign int other) const {
    //...
}
```
